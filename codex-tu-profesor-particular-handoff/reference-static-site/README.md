# Tu Profesor Particular — versión corregida HTML/CSS/JS

Esta carpeta contiene dos formas de usar la web:

1. `index-autonomo-completo.html`  
   Archivo único con HTML + CSS + JS + logo/fotos embebidos. Es el más seguro para abrir directo o pegar en plataformas online donde suelen romperse las rutas de imágenes.

2. `index.html` + `styles.css` + `script.js` + carpeta `assets/`  
   Proyecto separado en archivos limpios. Para que funcione, hay que subir la carpeta completa, no solo el HTML.

Notas:
- Se tomó como fuente `landing/index.html` del ZIP original.
- Se conservaron animaciones, interacciones, botones, selectores, FAQ, drawer mobile, progress bar y scripts originales.
- No se incluyeron DNI ni CV del ZIP original.
- Imágenes usadas: 8 archivos del sitio original.
