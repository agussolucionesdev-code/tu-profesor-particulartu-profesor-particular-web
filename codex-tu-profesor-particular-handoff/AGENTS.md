# AGENTS.md — instrucciones para agentes de código

## Prioridad del proyecto
Este repo pertenece al emprendimiento educativo **Tu Profesor Particular** de Agustín Elías Sosa.

La landing estática ubicada en `reference-static-site/` es la fuente de verdad. Cualquier migración o refactor debe mantener fidelidad visual y funcional antes de agregar mejoras.

## Reglas para trabajar
1. Leer primero `00_LEEME_PRIMERO.md` y `01_PROMPT_MAESTRO_CODEX.md`.
2. No eliminar assets ni cambiar nombres sin actualizar referencias.
3. No rediseñar la identidad visual sin pedido explícito.
4. Mantener CTAs a WhatsApp visibles y funcionales.
5. No subir credenciales ni archivos privados.
6. Ejecutar build antes de cerrar una tarea.
7. Dejar notas claras en README o docs cuando algo quede pendiente.

## Criterios de calidad
- Diseño fiel a referencia.
- Componentes React claros y mantenibles.
- CSS ordenado por tokens/componentes.
- Backend mínimo seguro.
- Deploy simple en Vercel y Render.
